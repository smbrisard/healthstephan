# Health Stephan — Free Hosting Guide

## 3 completely free options (no credit limits)

---

### Option 1: GitHub Pages (recommended, free forever)
1. Go to **github.com** → sign up free
2. Click **+** → **New repository** → name it `healthstephan` → Public → Create
3. Click **uploading an existing file** → drag all 4 files + icons folder → Commit
4. Go to **Settings** → **Pages** → Branch: main / root → Save
5. Your URL: `https://YOUR-USERNAME.github.io/healthstephan`

### Option 2: Vercel (free, instant)
1. Go to **vercel.com** → sign up free
2. New Project → Browse → upload the HealthStephan-PWA folder → Deploy
3. Get instant URL

### Option 3: Cloudflare Pages (free, fastest CDN)
1. Go to **pages.cloudflare.com** → sign up free
2. Create project → Direct Upload → upload folder → Deploy

---

## Install on iPhone
Open the URL in **Safari** → Share button → Add to Home Screen → Add
